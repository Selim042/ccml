# Horizontal Rule

> Back to [tags](./)

Prints a line across the view horizontally.  If supplied a pattern, will repeat the pattern.

| Attributes | Required | Default | Value                    |
|------------|----------|---------|--------------------------|
| pattern    | False    | "-"     | String to be repeated    |

## Example

```xml
<ccml>
  <body>
    <hr/><br/>
    <hr pattern="-+-"/>
  </body>
</ccml>
```
