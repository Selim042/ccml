# Image

> Back to [tags](./)

Renders the supplied image.

### Supported image types:

- [NFP](https://tweaked.cc/module/paintutils.html)

---

| Attributes | Required | Default | Value         |
|------------|----------|---------|---------------|
| src        | True     |         | Path to image |

## Example

```xml
<ccml>
  <body>
    <img src="image.nfp"/>
  </body>
</ccml>
```
