# Align
> Back to [tags](./)

Tag used to align child tags to center or right of the parent element.

| Attributes | Required | Default | Values        |
|------------|----------|---------|---------------|
| value      | True     |         | center, right |

## Example
```xml
<ccml>
  <body>
  <align value="center"><text>Hello, this is centered text!</text></align>
  </body>
</ccml>
```
