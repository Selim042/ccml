# Link

> Back to [tags](./)

Provites a clickable link to another page.

| Attributes | Required | Default | Value        |
|------------|----------|---------|--------------|
| dest       | True     |         | Path to page |

## Example

```xml
<ccml>
  <body>
    <text>Click</text><link dest="other-page.ccml"><text>here!</text></link>
  </body>
</ccml>
```
